iter( obj[, sentinel] )

Creates an iterator object.

If only a single argument is given, the given object must support the iteration protocol or the sequence protocol. A TypeError is raised if it does not support either of these.

If a second argument is given, the first argument must be a callable object. The iterator will call it will no arguments for each call to its own next method. StopIteration will be raised when the value returned is equal to the second given argument.
