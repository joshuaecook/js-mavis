itertools.js

Functions

itertools.chain( iterable1[, iterable2 …] )

Creates an iterator that treats several, consecutive iterable sequences as a single sequence.

itertools.count( n )

Creates an interator that produces an infinite arithmetic progression starting with n.

itertools.cycle( iterable )

Creates an iterator that repeatedly, indefinitely returns the elements from an iterable sequence.

itertools.ifilter( predicate, iterable )

Creates an iterator that returns the elements of an iterable sequence for which a predicate holds true.

itertools.imap( f, iterable1[, iterable2, …] )

Creates an iterator that computes the result of applying f to each item in the given iterable sequence.
